Recent Projects

Front-End

Cocktail Index App

Client-side cocktail-app using data from an external API. Users see a list of cocktails and filter by name using a search box. Details about the cocktail, like what ingredients and type of glass are needed, how to mix it and an example photo, can be obtained when opening the modal by clicking on the "How to mix it" button.

Challenges: Extract the necessary information from an exisiting external API, targeting the right endpoints and using the right HTTP protocols. Implementing Bootstrap and making it work within the boundaries of Bootstrap.

Key steps: Connecting the app to the external API. Figuring out how to access the information. Making the app userfriendly and responsive to all screen sizes. Running test for browser accessibility and using polyfills for fetch and promise. Text about the Web App

MyFlix App

This is the client side interface based on a previously designed API (Movie API). The MyFlix App lets users register and create their accounts, login, view and filter the movie database and create a favorite movies list. The information available about the movies are the plot, genre, main actors, and information about the director.

Challenges: Build the user interface according to the API endpoints, getting acquainted with React, implementing Redux boilerplate code, deploying to Netlify and writing a .toml file

Key steps: Setting up React app from scratch, connecting app to external API, creating components and implementing Routing and Redux (reducers and actions)

Tools:

Parcel js, Scss, React, Redux, Bootstrap, Netlify

Full-Stack (Back-End)

Movie API

This is the server side app for the MyFlix App. It provides a database about movies and several endpoints to interact with the data via CRUD operations.

Challenges: Build RESTful API from scratch. Defining and building database with Mongoose and MongoDB

Key steps: Setting up databases with two collections (User & Movies) through MongoDB. Deploying database to MongoDBAtlas (DBaaS) and API to Heroku (PaaS). Setting up data security: employing HTTP authentication and generating JWT tokens through Passport, hashing passwords with Bcrypt, and validating data input with express validator

Tools:

MongoDB, MongoDBAtlas, Mongoose, Heroku
